# Python Week 2 - Day 4

## Contact Book CLI Application

### Overview

This repository contains my Day 4 mini project, **Contact Book CLI Application**, developed using Python. The application allows users to manage contacts through a command-line interface (CLI). It demonstrates the practical use of Python programming concepts learned during Week 2.

## Features

- Add a new contact
- View all contacts
- Search for a contact
- Delete a contact
- Save contacts to a text file
- Load saved contacts when the program starts

## Topics Covered

- Variables
- Input and Output
- If-Else Statements
- While Loop
- For Loop
- Functions
- Dictionaries
- String Methods
- File Handling (File I/O)
- Exception Handling

## Files Included

- `project.py` – Main Contact Book CLI application
- `contacts.txt` – Stores contact information
- `README.md` – Project documentation

## Learning Outcomes

After completing this project, I learned how to:
- Build a complete CLI application.
- Store and retrieve data using File I/O.
- Organize code using functions.
- Handle errors using exception handling.
- Apply multiple Python concepts in a real-world project.
